Cellc

URI https://w3id.org/dggs/as/Cell
Source http://www.opengis.net/doc/AS/dggs/2.0
Description

A Cell is a Feature that has a geometry that is defined by a DGGS
Usage Note A DGGS cell can be considered a container for storing and retrieving data within a DGGS implementation. A DGGS cell may be considered either as a parent cell or as a child cell of at least one parent. In different DGGS implementations this container may be explicit (i.e. tightly coupled to the data stored on disc) or virtual (i.e. loosely coupled to the data stored on disc through a lookup table or database).
Example 
                      # example showing a Cell's Geometry and indicating the Geometry's conformance to a DGGS dataset
tb16pix:R222 a dgas:Cell ;
  skos:prefLabel "Cell R222" ;
  geo:hasGeometry [
    dcterms:conformsTo <https://w3id.org/dggs/tb16pix> ;
  	geox:asDGGS "R232"^^geo:dggsLiteral ;
  ] ;
.

Discrete Global Grid Systemc

URI https://w3id.org/dggs/as/DGGS
Source
Description

A Discrete Global Grid System is an integrated system of hierarchical Grids comprised of Cells.
Example 
                      # the dataset declaration for the "Testbed 16 Pix" DGGS
tb16pix:Resolution3 
	a dgas:DGGS ;
	skos:prefLabel "Testbed 16 Pix" ;
	skos:altLabel "TB16Pix" ;
	skos:definition "A Discrete Global Grid, based on rHealPix, with all Level 0 Cell corners placed in oceans. The rHealPix Ellipsoid parameters for this dataset are: a=6378137.0, b=6356752.314140356, e=0.0578063088401, f=0.003352810681182, lon_0=-131.25 and the RHEALPixDGGS constructor parameters are: ellipsoid=as above, north_square=0, south_square=0, N_side=3. " ;
	dcterms:conformsTo <https://w3id.org/dggs/as> ;
.

Gridc

URI https://w3id.org/dggs/as/Grid
Source http://www.opengis.net/doc/AS/dggs/2.0
Description

A network of Cells with geometries of equal area and shape that cover the entire Earth
Example 
                      # a Grid from the TB16Pix DGGS dataset
tb16pix:GridR3 
	a dgas:Grid ;
	skos:prefLabel "Gid of Resolution 3" ;
	skos:altLabel "Grid R3" ;
    skos:definition "TB16Pix DGGS Grid with resolution 3 which is the 3rd-order child of the coarsest TB16Pix Grid" ;
	dgas:hasCoarserGrid tb16pix:GridR2 ;
	dgas:hasFinerGrid tb16pix:GridR4 ;
.

Zonec

URI https://w3id.org/dggs/as/Zone
Source
Description

A particular region of space-time
Usage Note A Zone is a DGGS' Cells' Geometry. The extent of the geometry must be described by the Zonal Identifier literal value of the geometry for the DGGS in which the Geometry is defined. It may optionally be described by any number of other DGGS or other geometry literal values.
Super-classes geo:Geometryc
